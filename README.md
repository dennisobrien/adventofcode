# Tools

## Jupyter

Run the jupyter server locally with:

```
uv run jupyter lab \
    --notebook-dir=/home/dennis/workspace/dennisobrien/ \
    --ServerApp.allow_origin='*' \
    --ServerApp.ip='0.0.0.0'
```

See the project `dennisobrien/jupyter-uv` for more details.
